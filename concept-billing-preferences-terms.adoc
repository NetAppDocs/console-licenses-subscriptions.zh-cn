= 术语和定义 :hardbreaks: :nofooter: :icons: font :linkattrs: :imagesdir: ./media/

[role="lead"]
计费首选项控制NetApp Console如何处理您的许可证和订阅的付款、发票和续订行为。您可以在以下两个级别之一设置偏好：

* *全球* — 适用于直接许可、市场订阅和Keystone订阅。
* *按订阅计费* — 仅适用于特定市场或直接许可合同。


账单来源可以是以下几种：

* *市场订阅* — 计费通过 AWS Marketplace、Azure Marketplace 或 Google Cloud Marketplace 进行。您可以选择云市场提供的计费帐户、地区和续订选项。
* *直接许可* — 计费通过NetApp进行。您可以在NetApp Console中管理许可证密钥、发票联系人和自动续订。
* * Keystone订阅* — 账单将通过您的Keystone协议进行结算。您可以在NetApp Console中管理发票收件人和成本中心标签。


== 权限 您需要拥有 *帐户管理员* 或 *账单管理员* 角色才能更改账单首选项。操作员可以查看偏好设置，但无法修改。

== 设置账单偏好

计费偏好设置可让您： - 在所有服务中强制使用单一发票联系人列表， - 将使用情况与成本中心对齐， - 管理自动续订和到期通知， - 确保新的Cloud Volumes ONTAP (CVO) 系统继承正确的计费帐户。

= 账单字段：页面角色：参考

== Marketplace 字段 * *Marketplace 提供商* — AWS、Azure 或 Google Cloud。  **结算账户/订阅 ID** — 用于收费的云结算账户。  **合同期限** — 期限长度和续约行为。

= 计费选项和定义：页面角色：参考

== 条款和选项 * ** NetApp许可证优先 ** — 根据有效的NetApp许可证计费。更多使用权转移到了市场。  * **仅限市场订阅用户** — 所有使用量均通过市场合同计费。  * **按需付费（PAYGO）** — 按实际使用量计费的灵活订阅模式。  * **年度合同** — 固定期限的市场订阅。  * **自动续约** — 合同到期后自动续约。  * **自定义 CVO 配置** — 高级模式，用于映射每个超大规模数据中心的多个订阅。  * **直接许可** — 在市场容量之前使用的NetApp管理许可。  * **超大规模云服务提供商** — AWS、Azure 或 Google Cloud 提供商。  * **确认更改对话框** — 在应用更新之前显示，以防止意外更改账单。

== 典型配置路径 | 场景 | 推荐计费选项 | |-----------|-----------------------------| | 使用购买的NetApp容量 | 优先使用NetApp许可证 | | 仅使用 Marketplace 合约 | 仅使用 Marketplace 订阅 | | 在同一云提供商下为多个项目计费 | 启用自定义 CVO 配置 | | 恢复默认计费 | 禁用自定义 CVO 配置 |
