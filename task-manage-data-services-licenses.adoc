= 管理NetApp数据服务的许可证
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以管理直接从NetApp (BYOL) 购买的许可证，以用于NetApp数据服务，包括Cloud Volumes ONTAP。您可以查看已使用的许可证容量、剩余的可用容量，并且当达到容量限制或到期日期时您会看到通知。


NOTE: *直接许可*页面列出了所有许可证。如果您想要特定数据服务的许可证详细信息，请使用*概览*仪表板上的数据服务图块。link:task-homepage.html#overview-page["详细了解概览仪表板"] 。

此页面上的说明提供了适用于每项服务的信息。有关这些服务许可的更多具体信息，请参阅以下页面：

* https://docs.netapp.com/us-en/bluexp-backup-recovery/br-start-licensing.html["设置BlueXP backup and recovery的许可"^]
* https://docs.netapp.com/us-en/bluexp-disaster-recovery/get-started/dr-licensing.html["设置BlueXP disaster recovery的许可"^]
* https://docs.netapp.com/us-en/bluexp-ransomware-protection/rp-start-licenses.html["设置BlueXP ransomware protection的许可"^]
* https://docs.netapp.com/us-en/bluexp-tiering/task-licensing-cloud-tiering.html["设置BlueXP tiering许可"^]
* https://docs.netapp.com/us-en/bluexp-cloud-volumes-ontap/concept-licensing.html["设置Cloud Volumes ONTAP许可"^]
