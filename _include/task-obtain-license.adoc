如果控制台无法访问互联网（私人模式安装），您应该获取NetApp许可证文件进行上传。

当您从NetApp销售代表处购买许可证时，您会收到一封包含序列号和许可详细信息的电子邮件。如果许可证未自动出现在控制台中，请使用序列号获取许可证文件。

.步骤
. 查找您的控制台帐户 ID：
+
.. 在控制台的右上角，选择image:icon-settings-option.png["显示在 Web 控制台右上角的设置图标。"]> *身份和访问管理*。
.. 在组织页面上，查找您的帐户 ID 并复制它。
+
如果没有列出帐户 ID，而您只有组织 ID，请复制组织 ID 的前八个字符并将其附加到 *account-*

+
例如，如果这是您的组织 ID：

+
ea10e1c6-80cc-4219-8e99-3c3e6b161ba5

+
您的帐户 ID 为：

+
帐户-ea10e1c6



. Sign in https://mysupport.netapp.com["NetApp 支持站点"^]并选择*系统 > 软件许可证*。
. 输入您的许可证序列号。
. 在*许可证密钥*列中，选择*获取NetApp许可证文件*。
. 输入您的控制台帐户 ID（在支持站点上称为租户 ID）并选择 *提交* 以下载许可证文件。
